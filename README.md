# Ahbap Tekirdağ - Şubat 2026 Etkinlik Takvimi

Ahbap Tekirdağ gönüllülerinin Şubat 2026 etkinliklerini gösteren interaktif web takvimi.

## Etkinlikler

| Tarih | Etkinlik | Açıklama |
|-------|----------|----------|
| 1 Şubat | Kitap Arkadaşım Buluşması | Küçük Prens kitabı okuma ve sohbet |
| 8 Şubat | Gönüllülerle Tanışma | 6 Şubat Depremi anma ve AFAD eğitimi |
| 15 Şubat | Dünya Kediler Günü | Sokak hayvanlarına yuva yapımı ve mama dağıtımı |
| 21 Şubat | Sevgiye İlmek | İftar sonrası örgü etkinliği |
| 28 Şubat | Quiz Night | Emojilerle şarkı tahmin ve oyun gecesi |

## Özellikler

- Responsive tasarım (mobil, tablet, masaüstü uyumlu)
- Animasyonlu kalp arka planı
- Sayfa açılış animasyonları
- Etkinlik detayları için modal pencere
- iOS ve Android touch desteği

## Kullanım

Takvimi görüntülemek için: [[https://asyamirzaoglu.github.io/ahbap-subat-takvim/](https://asyamirzaoglu.github.io/ahbap-subat-takvim/)](https://ahbap-tekirdag.github.io/ahbap-subat-takvim/)

## Teknolojiler

- HTML5
- CSS3 (Animasyonlar, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Poppins)
